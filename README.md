ML Kit를 활용한 실시간 스쿼트 자세 감지 Flutter 애플리케이션입니다. 카메라를 통해 사용자의 자세를 실시간으로 분석하고 피드백을 제공합니다.

## ✨ 주요 기능

- 📸 실시간 포즈 감지
  - ML Kit를 활용한 고성능 자세 인식
  - 부드러운 실시간 처리

- 🎯 스쿼트 자세 분석
  - 정확한 관절 각도 계산
  - 단계별 자세 검증

- 💡 시각적 피드백
  - 직관적인 자세 가이드라인
  - 실시간 오버레이 표시

- 🔢 스쿼트 카운터
  - 정확한 횟수 측정
  - 자동 카운팅 시스템

## 🚀 기술 스택

- Flutter
- Google ML Kit
- Camera Plugin
- Custom Painter
