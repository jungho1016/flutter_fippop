# Flutter Squat Pose Detection

ML Kit를 사용한 실시간 스쿼트 자세 감지 Flutter 애플리케이션

## 기능

- 실시간 포즈 감지
- 스쿼트 자세 분석
- 시각적 피드백
- 스쿼트 카운터

## 시작하기

1. 필수 조건:
   - Flutter SDK
   - Android Studio 또는 VS Code
   - iOS/Android 개발 환경

2. 설치:
   ```bash
   flutter pub get
   ```

3. 실행:
   ```bash
   flutter run
   ```
